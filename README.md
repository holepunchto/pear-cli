# pear

Bootstrap Pear Runtime

```
npx pear
```

That's it!

Help: `pear`, `pear help [cmd]`, `pear [cmd] -h`

Documentation: [https://docs.pears.com](https://docs.pears.com)

Pear Runtime: [https://github.com/holepunchto/pear](https://github.com/holepunchto/pear)
