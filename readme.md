## Getting Started

### Install:

```sh
npm install -g @holepunchto/pear
```

### Bootstrap

```sh
pear
```

First run will ask to bootstrap. There is also the bootstrap command:

```sh
pear bootstrap
```

Without UI runtime functionality:

```sh
pear bootstrap --nongui
```

### Help

**General**

```sh
pear help <cmd>
```

```sh
pear <cmd> --help
```

```
pear <cmd> -h
```
